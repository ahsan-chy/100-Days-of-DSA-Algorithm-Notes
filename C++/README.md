### Void Main - Int Main
Void means nothing.
- The **`void sum()`** indicates that the sum() function **will not return any value,**.

- **`int main()`** indicates that the main() **can return integer type data.**

- **`long main()`** indicates that the main() **can return long type data.**

.

#### MAX_INT

#### MIN_INT
```c++

```
